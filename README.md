## ПрофДент — сайт стоматологии (Ступино)

Многостраничный статический сайт (HTML/CSS/JS) в стиле промо-сайта стоматологической сети: главная, клиника, специалисты, оборудование, цены, контакты, блог.

### Быстрый запуск

- **Вариант 1 (просто открыть)**: откройте `index.html` в браузере.
- **Вариант 2 (локальный сервер, рекомендуется)**:

```bash
python3 -m http.server 5173
```

Затем откройте `http://localhost:5173`.

### Страницы

- `index.html` — главная
- `clinic.html` — клиника
- `specialists.html` — специалисты
- `equipment.html` — оборудование
- `prices.html` — цены
- `contacts.html` — контакты (Ступино)
- `blog.html` — блог
